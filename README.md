# Excelent Images

## What

You present it with an image file and it bestows upon you an Excel file (as if by magic).

## How

Each pixel is translsated to a spreadsheet cell.

## Usage

1. Add image to root
2. Set image name in script
3. Run it:

```bash
cargo run
```
